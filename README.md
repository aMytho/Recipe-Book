

## Description

A recipebook built with [NestJS](https://github.com/nestjs/nest), [Angular](https://angular.io/), and [MongoDB](https://www.mongodb.com/).

## Installation

```bash
$ npm install
```

```bash
# Frontend Deps
$ cd frontend
$ npm install
```

## Running the app

```bash
$ npm start
# Frontend
$ cd frontend
$ npm start
```
